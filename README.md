# srikarvardhan.me

Personal portfolio of Mangadoddi Srikar Vardhan, AI/ML engineer and researcher.

Static multi-page site (HTML + CSS, no build step), served via GitHub Pages on the custom domain [srikarvardhan.me](https://srikarvardhan.me).

## Pages
- `index.html` — home
- `projects.html` — shipped projects
- `research.html` — peer-reviewed publications
- `services.html` — freelance / work with me
- `about.html` — bio, experience, contact

## Local preview
```
python -m http.server 5500
```
Then open http://localhost:5500
